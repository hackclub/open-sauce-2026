# Starting your project

Starting your project is super simple! Here's what you need to do:

_before anything, make sure to join [#outpost](https://hackclub.enterprise.slack.com/archives/C0B04RP43TQ) in the slack!_

## 1. Get an idea!

The first step is to come up with an idea for a project! Here's some cool inspo sources:

- [Adafruit Learn](https://learn.adafruit.com/) has an awesome list of projects!
- [Hackaday](https://hackaday.com) often posts interesting info!
- Watch some past Open Sauce Recaps and see what projects were there!

My advice is to think of something that would be really cool to bring with you and demo. Sketches help a ton!

## 2. Set up your project!

This involves a couple different steps; you'll need a:

- [GitHub](https://github.com/) repository! This is where your project will be stored and shared. _other git hosts are supported too!_
- a README.md file in the root to describe what your project is!
- a way to track your progress!
- a JOURNAL.md file that you'll update every time you work on your project!

## 3. Share what you're making in slack!

Write a short message in #outpost about what you're making! You should include a really basic sketch - don't worry if you're not an artist! It just helps establish a north star.

## 4. Start working away!

Start designing! While you're working on your project, make sure to write a journal entry every time you work on it! Here's an easy template for you to copy:

```Markdown
# May 4th, 2026: Title!

[your actual journal entry!]

[A picture or two of what you did!]

**Total time spent: 2h**
```

Next, you probably want to go to [designing your project](/info/design)
